# 💰 Expense Tracker (Python + SQLite)

## 📌 Description

This is a simple command-line Expense Tracker built using Python and SQLite.
It allows users to record, manage, and analyze their daily expenses.

---

## 🚀 Features

* Add new expenses
* View all expenses
* Update existing records
* Delete expenses
* Filter by category
* Filter by date
* Monthly total calculation
* Category-based summary

---

## 🛠️ Technologies Used

* Python
* SQLite
* datetime module

---

## ▶️ How to Run

```bash
python main.py
```

---

## 📂 Project Files

* `main.py` → User interface (menu system)
* `database_logic.py` → Database and logic
